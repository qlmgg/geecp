Gee Control Panel v1.0.1
===============

> 运行环境要求PHP7.1+。

## 主要新特性

* 支持插件机制
* 支持模板标签

## 支持的功能

* Hyper-v VPS主机
* 物理机租用


## 下载安装包

~~~
https://gitee.com/qiduocloud/geecp/repository/archive/v1.0.1?format=zip

## 配置服务器环境

~~~
推荐LNMP环境
Nginx （openresty 推荐+）
PHP 7.2+
Mysql 5.7 +

~~~
